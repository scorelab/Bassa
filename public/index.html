Automated Download Queue for Communities to take the best use of Internet bandwidth

Bassa solves the problem of wasting internet bandwidth by queuing a download if it is larger than a given threshold value in high traffic and when the traffic is low, it completes the download of the files. After the files are downloaded, the users can get their files from the local servers which do not require external internet bandwidth.

  • Provides an interface for users to add their downloads as links or torrent magnet links

  • Provide users with an interface to view and download the files in the local server

  • Provide a rating system for users to rate the files residing in local server.

  • Automatically start and stop downloading in given time frame

  • Automatically clean the disks and make room for new downloads

  • Notify the users when his/her download is completed

  • Mark inappropriate downloads

  • Provides admins with an interface to deal with inappropriate files

Install Bassa

INSTALLATION GUIDE

  • Docker
  • MacOS
  • Linux
  • Windows OS

Setup Bassa using Docker (Recommended)

Install and Setup Docker, Docker Compose and git on your machine. Run this command to check if you have installed properly.
docker -v && docker-compose -v && git --version

Download Bassa git repository
git clone https://github.com/scorelab/Bassa && cd Bassa

Start Bassa
docker-compose up --build
You might need to wait for several minutes for the first time installation but later on startups would be faster. Now go to http://localhost:3000 and you should be able to use Bassa locally.

Setup Bassa manually on MacOS machine

Download Bassa git repository
git clone https://github.com/scorelab/Bassa && cd Bassa

Install dependency software to run Bassa locally.
sudo ./setup.sh

Install python modules for Bassa API server
cd components/core && python3 setup.py develop

Back to the root folder: Start MySQL and create a table named Bassa
brew services start mysql
mysql -u root -p
mysql>> Create Database Bassa;
mysql>> exit

Insert database schema
cd db_schema && mysql -u root -p Bassa < Bassa.sql
